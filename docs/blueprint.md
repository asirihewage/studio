# **App Name**: PhotoFake

## Core Features:

- Image Upload: Allow users to upload image files to the application.
- EXIF Data Display: Display the existing EXIF metadata of the uploaded image.
- Location Selection: Allow the user to search and select a new location (latitude, longitude) using a map interface.
- Timestamp Adjustment: Provide an interface to modify the date and time the photo was supposedly taken.
- AI-Powered Realistic EXIF Generation: Using a generative AI tool, cross-reference phone model, time, date, and location to create realistic and consistent EXIF data. If the user does not choose a specific phone, the tool should suggest one to the user to continue, instead of crashing.
- Metadata Application: Apply the new or modified EXIF data to the image, updating its metadata.
- Image Download: Allow users to download the image with the modified EXIF data.

## Style Guidelines:

- Primary color: Soft blue (#7BB4E6) for a clean and trustworthy feel.
- Background color: Very light gray (#F0F4F7) for a subtle and professional look.
- Accent color: Teal (#46A5A5) to highlight key actions and interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look.
- Use simple, clear icons to represent different metadata fields and actions.
- Maintain a clean, organized layout with clear sections for image upload, metadata display, and modification options.
- Use subtle transitions and animations to provide feedback on user interactions.
